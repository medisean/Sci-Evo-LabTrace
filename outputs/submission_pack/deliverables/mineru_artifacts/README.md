# MinerU Artifacts

本目录包含 4 篇开放许可论文的 MinerU 结构化解析产物，用于评审复核 evidence 字段与原文之间的对应关系。

每个 case 子目录包含：

- `full.md`: MinerU 转换后的 Markdown 文本。
- `content_list.json`: 页面与块级内容列表。
- `content_list_v2.json`: 结构化内容列表 v2。
- `layout.json`: 页面布局信息。
- `model.json`: 模型解析结果。

不包含 API token、下载 zip、未确认许可的赛事样例全文解析产物或图片目录。
